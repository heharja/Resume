1. Double click setup.exe file path:"D:\drivers\Audio\setup.exe"

2.It pops-up "User Account Control" window, press "Yes" button to continue.

3. System pops-up "Realtek High Definition Audio Driver Setup" window, press "Next" button to install driver.

4. Driver will be installed for a few minutes.

5. When the Installation completed to install driver, press "Finish" button to reboot the computer.
